    Pult

    Static File Server

    A simple static file HTTP server that the Pult command line runs for the --static option. Used to easily make static assets available on a local .dev domain.

    This could be extended in future versions to serve directory listings as well as files.

    Write Status

    Write an HTTP status code and its status text to the body of an HTTP response.

    function writeStatus(status, headers) {
  if (!headers) headers = {};
  this.logType = headers['Content-Type'] = 'text/plain';
  this.logLength = headers['Content-Length'] = http.STATUS_CODES[status].length;
  this.writeHead(status, headers);
  if (this.req.method != 'HEAD')
    this.write(http.STATUS_CODES[status]);
}
